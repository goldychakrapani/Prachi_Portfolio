A Live Weather App allows users to input a city name and get real-time weather data including temperature, humidity, wind speed, and weather condition icons. This app fetches data from a weather API and dynamically displays the result using JavaScript.

A Full Stack SignUp Project

This full-stack Register/Login/Logout Authentication System allows users to create accounts, securely log in, and log out. It uses HTML, CSS, and JavaScript on the frontend, with a Node.js + Express.js backend, and MongoDB for the database. Passwords are hashed using bcrypt, and user authentication is handled via sessions or JWT tokens. The app includes user input validation, protected routes that restrict access to logged-in users, and logout functionality that clears the session or token.
